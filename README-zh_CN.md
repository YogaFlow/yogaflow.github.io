# 如何离线浏览

你需要安装[Git](https://git-scm.com) 和 [Node.js](https://nodejs.org), 然后执行下边的指令:

```bash
mkdir yoga-flow && cd yoga-flow
git clone https://github.com/YogaFlow/yogaflow.github.io.git
yarn
yarn run serve
```

在一个新的浏览器标签页打开
